# StayX-Smart-Hostel-PG-Finder
StayX is a single-page web application designed to help students and working professionals explore hostel and PG accommodation options based on essential living factors such as food quality, Wi-Fi, safety, hygiene, rent affordability, study environment, and visitor policies.

This project is inspired by academic research focusing on user needs, decision-making factors, and satisfaction levels, enabling a digital, transparent, and user-driven accommodation selection system.
